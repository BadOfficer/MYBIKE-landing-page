# MYBIKE - Landing page
<strong>MyBike</strong> is a responsive landing page for an electric bike brand that showcases products, highlights key features, and provides an easy way for users to explore bikes and contact the company.

## Live DEMO

Experience the live website:
[MYBIKE](https://BadOfficer.github.io/MYBIKE-landing-page/)

## Design reference

This project is based on the following Figma design:
[FIGMA](https://www.figma.com/file/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0%3A1)

## Technologies used
- HTML5
- SASS (SCSS)
- JavaScript

## Getting Started
1. Clone the repository:
```
git clone https://github.com/BadOfficer/MYBIKE-landing-page.git

cd MYBIKE-landing-page
```
2. Install dependencies:
```
npm install
# or
yarn install
```

3. Run the project locally:
```
npm start
# or
yarn start
```

## Features
- <b>Responsive design:</b> Optimized for different screen sizes and devices;
- <b>Hover effects:</b> Added hover effects for cards, links and buttons;
- <b>Animated sidebar:</b> Smooth slide-in sidebar that appears when clicking the menu button;
- <b>Sticky footer:</b> Sticky footer where an image smoothly slides in while scrolling the page;
